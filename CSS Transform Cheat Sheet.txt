============
#1. ROTATION
============

rotate(
  degValue,   => Rotation value in degree
  [x],        => [Optional] Rotation point x-origin
  [y]         => [Optional] Rotation point y-origin
)

  EQUIVALENT-TO

transform: rotate(degValue);
transform-origin: x y [z];
